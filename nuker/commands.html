Format Perms Description
|Shuffle Channels| <no args> MANAGE_CHANNELS Will randomly reorganize channels.
|Del Channels| <no args> MANAGE_CHANNELS Deletes all channels.
|Mass Create Channels| <name[rand] type[rand] num> MANAGE_CHANNELS Creates channels.
|Mass Edit Channels| <name topic slowmode nsfw> MANAGE_CHANNELS Edits channels.
|Mass Create Roles| <name[rand] num> MANAGE_ROLES Creates roles.
|Shuffle Roles| <no args> MANAGE_ROLES Will randomly reorganize roles.
|Del Roles| <no args> MANAGE_ROLES Deletes all roles.
|Mass Edit Roles| <name permissions(number)> MANAGE_ROLES Edits roles.
|Mass Nick| <nickname> MANAGE_NICKNAMES Changes everyone's nick.
|Mass Kick| <reason> KICK_MEMBERS Kicks everyone.
|Mass Ban| <reason> BAN_MEMBERS Bans everyone.
