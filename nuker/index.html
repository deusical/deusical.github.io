DiscordNuker is a nuker lib with electron gui - yes, it's open source. If you are just a user and want to see a list of commands, go to commands

The commands list works like this: |something| means the definition of the command is "something" and the [something] means something is a special arg.

If you are a coder who is here to learn, here's what to do.

(I'm assuming y'all have nodejs installed)

Open cmd and install necessary libraries with 

npm install

and run with

npm start

boom! You're done. The actual nuker module is in the nuker directory, if you want to take a look.
